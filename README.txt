1. Launch of application is made in Class Starter
(fasad/src/main/java/com/exposit/Starter.java);
2. Application is divided into 6 types of main menu.
3. Enter number of main menu and you will be directed to the bottom menu
with main activities of the application.
4. Data is saved in json format (utils/src/main/resources/).
5. After launching an application data will be automatically downloaded
from json files.
6. After making all needed manipulations with the data, perform save of
the data with the help of 'save' button of main menu.
7. Enter '0' to exit from the application.
